CSI 3335 Introduction to Databases

Microsoft Access Lab

Overview:

      Microsoft Access allows you to manage your information from a single database file. Within the file, you can divide your data into tables; view, add, and update table data using forms; find and retrieve the data you want using queries; and print data in a specific layout using reports.

        The lab consists of two sections. Section-A, takes you step by step through the process of creating an Access database with relationships, populate the database using forms, and extract data using SQL and reports. Section-B specifies an E-R diagram for which you will have to create a database with relationships, populate the database, and extract data.
