Creating a table in the Database

You can obtain a table two ways in Microsoft Access:

    ·         Use a wizard

    ·         Create a table from scratch

You can use any combination of features and techniques to design your table(s).  In this lab, we will describe the methods to create a table from scratch.

  • Once you have created an empty database, Access opens a dialog displaying the database objects. If this dialog box opens, click on Tables. On the dialog tool bar click New .

 

  

  • The New Table dialog opens. Access allows you to create a table in four different ways. In table Design view, you can create an entire table from scratch, or add to/delete from a table's fields. We’ll be using Design view to design our table.

  • Choose Design View and click OK. This will open up an empty table.
