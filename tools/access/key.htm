Specifying the Primary Key

  • From the main dialog window, click on the Student table. Then click on Design view . This will bring up the Student table in the Design view.

  • The Primary key for this table is SSN. Highlight the SSN entry, by clicking in the left most column of the attribute.

  • Click on the Primary Key on the tool bar. This sets the highlighted row as the primary key for the table.

  • Set the primary keys for the remaining tables.

·         Table: Course

Key: CID

·         Table: Professor

Key: PID

·         Table: Takes

Key: SSN, CID

·         Table: Taught

Key: PID, CID

 

  •  If a table has more than one key, highlight the rows containing the attributes and then choose the primary key option.
