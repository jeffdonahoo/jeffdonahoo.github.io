Usage
    
 
To use the machines, ssh as root to 129.62.151.140. Use port 22 (the default port) to get to virtual machine 1 and port 222 (-p 222) to get to virtual machine 2. Try getting out of the 192.168.1.0/24 network (without reconfiguring the router) just for fun. We recommend that you change the firewall password to something else before letting students loose inside this setup.
