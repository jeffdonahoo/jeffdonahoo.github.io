Creating a virtual machine
  
 
On a Windows host, use the Wizard for doing this. To do this, go to File >New > New Virtual Machine.
The New Virtual Machine Wizard presents you with a series of screens that you navigate using the Next and Prev buttons at the bottom of each screen. At each screen, follow the instructions, then click Next to proceed to the next screen. You can use the default options throughout. You will also be asked for the name of the guest OS you plan to install on this virtual machine. You should be able to find your OS on the drop down list of Operating Systems. For a more detailed discussion of what each option at each step means, go through the documentation at http://www.vmware.com/support/ws4/doc/new_guest_steps_ws.html .
