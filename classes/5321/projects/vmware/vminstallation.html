Installation on a Windows Host OS
Installation of VMWare on a Windows host is just like installing any other software program on Windows. You will need to log on as an administrator for installing VMWare Workstation. You will need the VMWare installation software(on a CD or a downloaded version) and a serial number before you start the install. Run the setup.exe file or a similar file(depends on the version of the software you are installing). Please look at http://www.vmware.com/support for the latest documentation on the product you are installing. The install we did was very simple and was the ‘Next, Next..Finish’ kind.

See here for install instructions on a Linux Host OS
http://www.vmware.com/support/linux/doc/install_config_linux.html .
