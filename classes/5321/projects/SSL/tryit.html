Try it yourself

 

 

 

For those who believe in learning by programming here are a few tips:

 

If your are using J2SDK 1.4 use the HttpsURLConnection class in the javax.net.ssl package,

Alternately if you still live in the J2SDK 1.3 world,use the HttpsURLConnection class in the com.sun.net.ssl package.

 

Using this url connection class you can connect to a secure web site (like https://www.verisign.com) over the https protocol and feel proud of yourself.
