Packet Sniffing Tools

These tools facilitate the capture and visualization of network traffic. Passive recording and decoding of network packets is extremely useful in network and security troubleshooting and research. These free tools accomplish network visibility to varying degrees, and each has it's own unique way of presenting the packet data.

  • Ethereal
  • Ksniffer
  • Snort
