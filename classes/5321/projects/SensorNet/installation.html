Installation
Please do not follow the instructions that come with the mote kit. They are for an old version of TinyOS, and is really painful to install(and it doesn’t work correctly in the end anyway). Instead, go to http://webs.cs.berkeley.edu/tos/download.html for the latest download. We used the Windows install, and it is a ‘Next..Next..Finish’ type of install. It is pretty straightforward and should not cause any of the old manual installation problems. Just follow the instructions on the screen.

One word of caution though! Make sure there are no past traces of any TinyOS components, especially cygwin. Delete all traces before beginning the install. The install program is not yet smart enough to overcome this problem automatically.

At the end of the installation, perform a quick sanity check. In the cygwin shell, run the following
bash$ /opt/tinyos-1.x/tools/scripts/toscheck
You should no errors and warnings.

Congratulations, you have TinyOS all set up and ready to go!
