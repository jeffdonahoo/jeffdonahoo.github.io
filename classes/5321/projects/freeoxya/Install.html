1. Before installing, make sure the VM is in your path.
  2. Download FreeOxy and unzip it
  3. To start a FreeOxy agent, go to where you unzipped the application and execute RunFreeOxy.bat.
  4. Set your browser to use the FreeOxy agent as a proxy with a port of 46411. Go here for proxy configuration instructions for IE. You can change the defaults by altering the FreeOxy parameters in RunFreeOxy.bat.

    java FreeOxy [client port] [msg port] [thread num] <default ttl>

    [client port] the port number for browser
    [msg port] the port number for FreeOxy peers
    [thread num] the size of the thread pool
    < default ttl> optional, the default ttl for request message
    e.g., java FreeOxy 46411 46412 12 3
