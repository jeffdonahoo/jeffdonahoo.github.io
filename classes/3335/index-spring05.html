        Program 1-Stage 1 Test

        One of key ways to test code in industry is with Unit Testing. Basically, Unit Testing applies a series of tests to each method. The standard Unit Test library in Java is JUnit (link only provided for the curious). I have written a JUnit Test for you, compiled it, and place it in a library (JAR) file. This is a blind test because you don't get to see the actual test code; however, the text of a test failure may give you some hints about the problem. You can execute a text or GUI version of the test driver. Either version will either signal success or give you the errors for the tests that failed.

        • Compile your StringToStringChainLinkedList.java
        • Download junit.jar and Test.jar to the same directory as StringToStringChainLinkedList.class
        • Execute the test using the text or GUI test driver:
          1. Text: java -cp "junit.jar;Test.jar;." junit.textui.TestRunner TestStringToStringChainLinkedList
          2. GUI: java -cp "junit.jar;Test.jar;." junit.swingui.TestRunner TestStringToStringChainLinkedList
        • Evaluate results. Look for:
          1. Text: OK (6 tests)
          2. GUI: Errors: 0 Failures: 0
      • Final Testing

        Program 1 Test

        JBuilder integrates JUnit so we will do our final stage testing there. Note that JBuilder is just a wrapper so this works that same as the non-JBuilder test you ran for the first stage. To test your final submission for Program 1, do the following:

        1. Download the JBuilder project zip file and unzip it.
        2. Create and place your StringToStringChainLinkedList.java and StringToStringHashTable.java in the src directory.
        3. Open the JBuilder project.
        4. Right-click TestStringToStringChainLinkedList.java and select "Run Test" to run the first test. JBuilder will tell you what passed and failed. You can even use "Run Debug" to step through the tests. Pretty cool, eh?
        5. Right-click TestStringToStringHashTable.java and select "Run Test" to run the second test.
